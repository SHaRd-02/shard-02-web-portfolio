Introduction

HTML is a markup language used to create the structure and content of web pages. It consists of a set of elements that define the layout, text, images, multimedia, and interactive elements of a webpage. HTML5 is the latest version of HTML, introducing new features such as native support for video and audio playback, canvas for drawing graphics, and improved support for offline applications.

HTML documents are text files with a .html extension. They are composed of elements enclosed within opening and closing tags. Elements can have attributes that provide additional information about the element or modify its behavior. HTML documents are rendered by web browsers, which interpret the markup and display the content to users.

HTML5 introduces new semantic elements, such as <header>, <nav>, <section>, <article>, <footer>, which provide clearer structure and meaning to web content.
Here's a basic example:


<!DOCTYPE html> 
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, 
    initial-scale=1.0">
    <title>HTML5 Example</title>
</head>
<body>
    <header> </header>
    <main> </main>
    <aside> </aside>
    <footer> </footer>
</body>
</html>

Elements

HTML elements are the building blocks of web pages. They define the structure and content of a webpage by encapsulating different types of content.
Some common HTML elements include:

  • Headings: <h1> to <h6> for defining headings of varying sizes.
  • Paragraphs: <p> for defining paragraphs of text
  • Links: <a> for creating hyperlinks to other web pages or resources.
  • Images: <img> for embedding images into the webpage.
  • Lists: <ul>;, <ol>, and <li> for creating unordered and ordered lists.

<h1>This is a Heading</h1>
<p>This is a paragraph of text.</p>
<a href="https://www.example.com">This is a link</a>
<img src="image.jpg" alt="Image Description">
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Each element can contain content and other nested elements, allowing for the creation of complex webpage layouts.

Attributes

HTML attributes provide additional information about an element or modify its behavior. Attributes are added within the opening tag of an element and consist of a name and value pair.
Some common attributes include:

  • id: Used to uniquely identify an element within a document.
  • class: Used to apply one or more CSS classes to an element.
  • src: Specifies the URL of the resource to be embedded (e.g., image source).
  • href: Specifies the URL of the linked resource (e.g., hyperlink reference).
  • alt: Provides alternative text for an image.
  • title: Provides additional information about an element (e.g., tooltip text)

<img src="image.jpg" alt="Image Description">
<a href="https://www.example.com" title="Link Title">
This is a link</a>
<input type="text" id="username" 
class="input-field" placeholder="Enter your username">

Attributes enhance the functionality and accessibility of HTML elements.

Inputs

HTML input elements are used within forms to collect user input. HTML5 introduces several new input types, each designed for specific types of data entry.
Some commonly used input types include:

  • Text: <input type="text"> for single-line text input.
  • Password: <input type="password"> for password input (text is masked).
  • Email: <input type="email"> for email address input with validation.
  • Checkbox: <input type="checkbox"> for selecting multiple options.
  • Radio: <input type="radio"< for selecting a single option from a list
  • Submit: for submitting a form.

<input type="text" placeholder="Enter your name">
<input type="password" 
placeholder="Enter your password">
<input type="email" placeholder="Enter your email">
<input type="checkbox" id="checkbox1" 
name="checkbox1" value="value1">
<input type="radio" id="radio1" name="radio" 
value="value1">
<input type="submit" value="Submit">

Input types help to standardize user input and provide built-in validation for certain data types.

Buttons

HTML buttons allow users to trigger actions or submit forms within a webpage. There are several types of buttons, each with its own purpose:

  • <button>: A clickable button with customizable content and behavior.
  • <input type="button">: A button with customizable content and behavior.
  • <input type="submit">: A button used to submit a form
  • <input type="reset">: A button used to reset a form to its initial state.

<button onclick="alert('Hello, World!')"
>Click Me</button>
<input type="button" value="Click Me">
<input type="submit" value="Submit Form">
<input type="reset" value="Reset Form">

Buttons can be styled using CSS to enhance their appearance and usability.

Forms

HTML forms are used to collect user input and submit it to a web server for processing. Forms consist of one or more input elements enclosed within a <form> element. Common form elements include text fields, checkboxes, radio buttons, dropdown menus, and submit buttons

When a user submits a form, the data is sent to the server specified in the form's action attribute via an HTTP request. The server processes the data and may return a response to the user. Forms play a crucial role in user interaction on the web and are used for various purposes such as user registration, data submission, and search functionality


 <form action="/submit-form" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name" required>
<label for="email">Email:</label>
<input type="email" id="email" name="email" required>
<input type="submit" value="Submit">
</form>

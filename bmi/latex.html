Absolute Value Equations and Inequalities

Introduction

Absolute value brackets behave in one of two ways: if the expression inside is positive, they do nothing, but if it’s negative, then the absolute value brackets change the sign of the expression. When that is simply a number, it’s easy to see what the absolute value is:

\( |{-2.5}| = 2.5, |{7}| = 7 \)

When an expression involves a variable, though, the process is more complicated because for some values of \( x \) the expression will be positive, and for other values of \( x \) it will be negative.

To solve such problems, set up both situations: one in which the absolute value brackets do nothing (because the expression inside is positive) and the other where the absolute value brackets will change the sign of what’s inside.

Example Problem:

Solve for \( x: 2|4x-5|-3<13 \)

Solution:

You always start by isolating the absolute value brackets:

\( x: 2|4x-5|-3<13 \)

\(\underline{\textsf{                      }\textcolor{magenta}{+3}\textsf{  }\textcolor{magenta}{+3}}\)

\( x: 2|4x-5|\textsf{       }<16 \)

Divide by \( 2: \)

\(\frac{2|4x-5|}{2}<\frac{16}{2} \)

\(\frac{\cancel{2}|4x-5|}{\cancel{2}}=\normalsize|4x-5|< 8 \)

Now set up the two cases: Do Nothing & Change the Sign.

For the first, just rewrite the equation without absolute value brackets.

For the second, write the expression inside the absolute value brackets in parentheses and put a \( -1 \) on the outside. DO NOT change the direction the inequality faces.
